# 관상 앱 - 시작 가이드 (한국어)

한국 관상학을 영어권 사용자에게 소개하는 웹 앱이 완성되었습니다! 🎉

## 📱 완성된 기능

### ✅ 핵심 기능
- **사진 업로드**: 사용자가 자신의 얼굴 사진을 업로드
- **AI 분석**: OpenAI GPT-4o Vision API를 사용한 실시간 관상 분석
- **한국 전통 관상학**: 이마, 눈, 코, 입, 턱 등 5가지 주요 특징 분석
- **운세 점수**: 재물운, 건강운, 직업운, 애정운 (0-100점)
- **결제 시스템**: Stripe를 통한 $1 결제 처리
- **K-Pop 스타 예시**: 유명 연예인들의 관상 예시 제공

### ✅ 디자인
- 🎨 한국 전통 색상 테마 (빨강, 파랑, 금색)
- 📱 반응형 디자인 (모바일, 태블릿, 데스크톱)
- ✨ 현대적이고 세련된 UI
- 🎯 사용자 친화적인 인터페이스

## 🚀 빠른 시작 (데모 모드)

API 키 없이도 바로 테스트할 수 있습니다!

```bash
cd Physiognomy
npm install
npm run dev
```

브라우저에서 http://localhost:3000 열기

**데모 모드에서는:**
- 사진 업로드 및 UI 테스트 가능
- 샘플 관상 결과 확인 가능
- 전체 사용자 흐름 체험 가능
- API 키 불필요

## 🔑 실제 AI 분석 활성화하기

### 1단계: OpenAI API 키 발급

1. https://platform.openai.com/api-keys 접속
2. 회원가입 또는 로그인
3. "Create new secret key" 클릭
4. API 키 복사 (sk-로 시작)

### 2단계: 환경 변수 설정

`.env.local` 파일 생성:

```bash
cp .env.local.example .env.local
```

`.env.local` 파일에 API 키 추가:

```env
OPENAI_API_KEY=sk-여기에-발급받은-키-입력
```

서버 재시작:

```bash
npm run dev
```

이제 실제 AI 기반 관상 분석이 작동합니다!

## 💳 실제 결제 활성화하기 (선택사항)

### 1단계: Stripe 계정 생성

1. https://dashboard.stripe.com/register 접속
2. 무료 가입
3. Developers → API Keys 메뉴
4. 두 개의 키 복사:
   - Secret key (sk_로 시작)
   - Publishable key (pk_로 시작)

### 2단계: 환경 변수에 추가

`.env.local`에 추가:

```env
STRIPE_SECRET_KEY=sk_test_여기에-입력
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=pk_test_여기에-입력
```

### 3단계: 결제 코드 활성화

`app/page.tsx` 파일의 33번째 줄 근처 주석 제거

서버 재시작 후 실제 결제가 처리됩니다!

## 🌐 배포하기

### Vercel (가장 쉬움 - 2분 소요)

1. GitHub에 코드 푸시
2. https://vercel.com 접속
3. "New Project" 클릭
4. 저장소 import
5. 환경 변수 추가:
   - `OPENAI_API_KEY`
   - `STRIPE_SECRET_KEY`
   - `NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY`
6. "Deploy" 클릭

완료! 앱이 실제로 배포됩니다! 🎉

## 🐛 문제 해결

### 서버가 시작되지 않을 때

```bash
rm -rf node_modules package-lock.json
npm install
npm run dev
```

### API 호출이 작동하지 않을 때

- `.env.local` 파일 존재 확인
- API 키가 올바른지 확인
- 서버 재시작
- 브라우저 콘솔에서 에러 확인

### 이미지 업로드 문제

- 파일 크기 확인 (최대 10MB)
- JPG 또는 PNG 형식 사용
- 다른 사진으로 시도
