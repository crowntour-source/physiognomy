# 테스트 체크리스트

앱을 완전히 테스트하기 위한 체크리스트입니다.

## ✅ 시각적 테스트

### 메인 페이지
- [ ] 헤더가 한국 색상(빨강/파랑)으로 표시됨
- [ ] "관상 - Korean Face Reading" 제목 보임
- [ ] "Unlock the Secrets of Your Face" 섹션 보임
- [ ] 업로드 영역이 큼직하고 명확함
- [ ] K-pop 스타 예시가 4개 표시됨
- [ ] 하단에 3개의 특징 카드 보임
- [ ] 푸터가 표시됨

### 반응형 테스트
- [ ] 브라우저 창 크기 조정 시 레이아웃 정상
- [ ] 모바일 크기로 줄여도 깨지지 않음
- [ ] 버튼들이 터치하기 적당한 크기

## 📸 기능 테스트

### 사진 업로드
1. [ ] "Click to upload your photo" 영역 클릭
2. [ ] 파일 선택 창이 열림
3. [ ] 얼굴 사진 선택 (JPG, PNG 아무거나)
4. [ ] 사진이 화면에 표시됨
5. [ ] 사진 크기가 적절함 (너무 크거나 작지 않음)
6. [ ] "Analyze My Face - $1" 버튼 보임
7. [ ] "Change Photo" 버튼 보임

### Change Photo 버튼
1. [ ] "Change Photo" 클릭
2. [ ] 사진이 사라지고 업로드 영역 다시 나타남
3. [ ] 다른 사진 선택 가능

## 💳 결제 모달 테스트

### 모달 열기
1. [ ] "Analyze My Face - $1" 클릭
2. [ ] 어두운 배경과 함께 모달 팝업
3. [ ] 모달이 화면 중앙에 위치

### 모달 내용
- [ ] "Complete Payment" 제목
- [ ] "$1.00" 가격 표시
- [ ] "How it works" 설명 박스 (파란색)
- [ ] 3단계 설명 표시
- [ ] "Pay $1 with PayPal" 버튼 (파란색)
- [ ] "I've Paid - Analyze Now" 버튼 (빨강/파랑 그라데이션)
- [ ] "Cancel" 버튼

### PayPal 버튼 테스트
1. [ ] "Pay $1 with PayPal" 클릭
2. [ ] 새 탭에서 PayPal 페이지 열림
3. [ ] URL이 `https://paypal.me/hankyungglobal/1`
4. [ ] PayPal 페이지에 $1.00 표시
5. [ ] 앱 탭으로 돌아올 수 있음

### Cancel 버튼
1. [ ] "Cancel" 클릭
2. [ ] 모달이 닫힘
3. [ ] 업로드한 사진은 그대로 남아있음

## 🔮 분석 기능 테스트

### 기본 분석
1. [ ] "I've Paid - Analyze Now" 클릭
2. [ ] 모달이 닫힘
3. [ ] "Analyzing your facial features..." 메시지 표시
4. [ ] 회전하는 로딩 애니메이션 보임
5. [ ] 약 2-3초 후 결과 표시

### 결과 확인
- [ ] "Overall Reading" 박스 (빨강/파랑 배경)
- [ ] 전체 운세 텍스트 표시
- [ ] 6개의 상세 카드 (Forehead, Eyes, Nose, Mouth, Chin)
- [ ] 각 카드에 금색 테두리
- [ ] "Fortune Indicators" 섹션
- [ ] 4개의 프로그레스 바 (Wealth, Health, Career, Love)
- [ ] 각 바에 점수 표시 (70-95 범위)
- [ ] 바가 점수만큼 채워짐

### 랜덤성 테스트
1. [ ] "Change Photo" 클릭하고 같은 사진 다시 업로드
2. [ ] 다시 분석 진행
3. [ ] **다른 결과가 나오는지 확인** (중요!)
4. [ ] 점수가 다른지 확인
5. [ ] 3-4번 반복해서 매번 다른 결과 확인
